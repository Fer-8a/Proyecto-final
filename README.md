# Proyecto-final
Mi proyecto final de python pro(Kodland)
mi idea es:

Hacer una pagina web que diga datos de el medio ambiente y puedas subir una imagen en algun lugar y que te diga si puedes reciclar eso y si no que hacer con el. Para eso yo creo que voy a usar la libreria que usamos para la deteccion de imagenes que se llama yolov3 y hacerlo en visual studio code. Yo usaria los lenguajes css y html para la pagina web y python para darle funcionalidad osea como los datos random y eso 
lo pondria todo en una pagina web y con botones hasta arriba va a aver un apartado donde diga datos de el mdio ambiente que va a servir mediante una API que me de chat gpt.
Despus va a haber un apartado donde puedas subir una imagen de un basurero y te diga si ahi puedes reciclar cosas o no y si si que pudes reciclar en ese basurero. 

yo creo que con 30-40 minutos diarios acabo el proyecto en aproximadamente 1 semana
